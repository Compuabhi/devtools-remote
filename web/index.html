Works only in Chrome

DevTools Remote enables users to share their browser tabs,
so they can be debugged remotely via Chrome DevTools.

1. Install Chrome extension

The extension adds an action button, so the user can share access to a given tab at any time.

2. Click share button

When clicking the share button, the extension connects to Chrome and connects to our API where it gets a unique link, that's only known by the server and browser.

3. Share unique link

The unique link is copied to the user's clipboard and can be shared to the person that needs debugging access to the given browser tab.

Just open it in Chrome.

That's it!

DevTools Remote is an exploration of how cross-browser
remote debugging protocols could bring value to end-users.
